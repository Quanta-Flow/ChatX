## ✨ 功能点

- 📦 基于Ant Design X 构建
- ⚙️ 数据本地存储，杜绝隐私泄露问题
- 🐳 支持DeepSeek R1 思考过程展示
- 🐑 已支持调用 [ollama](https://ollama.com/) API

## 项目设置

### 安装依赖

```bash
$ pnpm install
```

### 开发环境

```bash
$ pnpm dev
```

### 生成构建

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
